Navigate Belgrade using public transport system like a pro!

Need a nearby bus stop? Wonder where a line will take you to? Guess no more!
An absolute necessity for both residents and visitors alike. 

Features: 

• Vehicle line/stop lists conveniently displayed on maps
• Line/stop search and lookup
• Favorite stops
• Nearby stops
• Optional USSD call info
• Timetables

This app does not represent official Belgrade transport (and related) companies, so the data included may not always be completely up-to-date.
In the same manner, the app cannot display any real-time vehicle position data as it simply does not have granted access.
